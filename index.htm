ARN is down.

ARN development takes too much of my almost non-existent free time.
Studies, work, life and 4 dev projects. All of that has become too much for me.
It's no joke when I say that I am collapsing under all this workload.
It's just too much. I can't handle it anymore.

Unfortunately I can't finance this project with donations only, there were too little.
To give you a cost breakdown: ARN development took around 500-600 hours in total so far (for the remake only).
This is equivalent to getting around 12.000 $ payment if this was an official job.
Now add the server costs to it which are 15 $ per month.

So how much did I get from donations, in total? After a whole year, since May 2014?
I got 42,19 $. Minus the PayPal fees. If you've read the last paragraph, do the math.
You can still donate if you can't help with development but want to help me and the project financially.

Working on ARN requires the dedication and time of a full-time job.

We really need to either get the financial problem solved or find another developer for ARN.
I know that many people are relying on it (ARN has 1.200 registered accounts at the time of writing this)
and I would rather want this to stay alive like all of you.
Therefore I am asking you to consult your friends: Do you know any developers who are interested in working on this project?

The requirements for ARN are quite high which is why it's so hard to find another developer:
  • You need to know node.js
  • You need to know Aerospike or at least another NoSQL database
  • You need to understand JavaScript
  • You need to understand JSON
  • You need to understand PHP
  • You need to understand nginx
  • You need to understand Regular Expressions
  • You need to know how to use git for version control
  • You need to know how to administer a Linux system from a terminal
  • You need to know how to do common performance optimizations (e.g. caching)
  • It's a bonus if you are familiar with DigitalOcean
  • It's a bonus if you are familiar with Slack
If you feel like you're up to the task please get in touch with me.
If you're not a developer yourself but know a friend who is, please lead him to this site and ask if he or she is interested.

-- Eduard Urbach, 4th May 2015
